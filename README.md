📺 Prime Video Analysis Dashboard 
Overview:

This project presents an interactive Power BI dashboard analyzing content available on Amazon Prime Video. It provides insights into movies and TV shows based on genres, ratings, release years, and geographic distribution, enabling data-driven understanding of streaming trends.

##Objectives
Analyze the distribution of movies vs TV shows
Identify top genres and content trends
Explore ratings and audience preferences
Understand content growth over the years
Visualize country-wise content availability

##Dashboard Features
-Content Trend Analysis – Shows release trends over years
-Genre Distribution – Highlights most popular genres
-Ratings Analysis – Breakdown of content ratings (PG, R, etc.)
-Country-wise Analysis – Content production by country
-Movies vs TV Shows – Comparative analysis
-Interactive Filters – Dynamic slicers for deeper insights

##Tools & Technologies
Power BI – Data visualization and dashboard creation
Excel / CSV Dataset – Data source
Data Cleaning & Transformation – Power Query

##Contains attributes like:
Title
Genre
Director
Cast
Country
Release Year
Rating
Duration
